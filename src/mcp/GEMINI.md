# MCP Gateway Layer

ส่วนที่ทำหน้าที่เป็นสะพานเชื่อมต่อระหว่าง Gemini CLI และเอเจนต์ภายในระบบ:
- **Protocol**: ดำเนินการตาม Model Context Protocol (stdio JSON-RPC 2.0)
- **Tool Mapping**: แปลงความสามารถของเอเจนต์ให้เป็น Typed Tools ที่โมเดลหลักเรียกใช้ได้
- **Orchestration**: ควบคุมจังหวะการรับ-ส่งข้อมูลระหว่างผู้ใช้และเอเจนต์ย่อย

**คำแนะนำสำหรับเอเจนต์:**
- การเปลี่ยนแปลงชื่อ Tool หรือ Parameters ต้องระบุภาษาและประเภทข้อมูลให้ชัดเจนตามสคีมา
- รักษาความปลอดภัยในการส่งผ่าน Context ระหว่างเครื่องแม่และเครื่องลูก
