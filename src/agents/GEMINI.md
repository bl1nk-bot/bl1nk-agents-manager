# Agent Management Core

ส่วนนี้คือหัวใจหลักในการจัดการเอเจนต์ของระบบ:
- **Registry**: จัดการสถานะและการลงทะเบียนเอเจนต์ที่โหลดมาจากหน่วยความจำ
- **Router**: ใช้ตรรกะอัจฉริยะในการเลือกเอเจนต์ที่เหมาะสมกับงาน (Routing Decision)
- **Executor**: รับผิดชอบการรันคำสั่งจริง และการจัดการวงจรชีวิตของงาน (Pending -> Running -> Done)
- **Creator**: สร้างไฟล์เอเจนต์ใหม่โดยแยกเป็น 2 ส่วน (.md และ .json) ตามสถาปัตยกรรมล่าสุด

**คำแนะนำสำหรับเอเจนต์:**
- เมื่อแก้ไข Logic การรันงาน ให้ตรวจสอบความสอดคล้องกับ `src/mcp/`
- เมื่อเพิ่มความสามารถใหม่ ให้ตรวจสอบที่ `AgentConfig` ใน `src/config.rs`
