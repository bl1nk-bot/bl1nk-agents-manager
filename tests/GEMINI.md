# System Verification Suite

ที่เก็บชุดการทดสอบระดับ Integration และ End-to-End เพื่อยืนยันเสถียรภาพของระบบ:
- **Integration Tests**: ทดสอบการไหลของข้อมูลระหว่าง Registry -> Router -> Executor
- **Protocol Tests**: ยืนยันความถูกต้องของการสื่อสารผ่าน MCP
- **Edge Case Coverage**: ทดสอบสถานการณ์ผิดปกติ เช่น ไฟล์คอนฟิกพัง หรือเอเจนต์ไม่ตอบสนอง

**คำแนะนำสำหรับเอเจนต์:**
- ยึดหลัก TDD: เขียนเทสในโฟลเดอร์นี้ก่อนเริ่มแก้โค้ดที่ซับซ้อน
- รักษา Code Coverage ให้มากกว่า 90% สำหรับฟีเจอร์หลัก
